docker - 端口映射出现错误

Nov 19, 2021
#docker

背景

遇到一个情况，在 docker 启动容器时，进行端口映射，出现以下错误

原因

根据网上的资料说，

docker服务启动时定义的自定义链 DOCKER由于某种原因被清掉。
重启docker服务及可重新生成自定义链 DOCKER。

解决

重启 docker 服务，再启动容器。
