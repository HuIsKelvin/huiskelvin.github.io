C++ - 实现单例模式

Sep 2, 2021
#C++#设计模式

1 概述

总体实现方式：将构造函数设为私有的 private，然后通过类的一个公共静态函数来生成实例。
并且，拷贝构造函数、移动构造函数、赋值运算符等也要设为 private（或者=delete）

几种实现方式：

  • 懒汉（非线程安全）
  • 懒汉+互斥锁 mutex（双重检查锁模式）（线程安全）
  • 饿汉（天生线程安全）
  • 局部静态实例（线程安全）
  • pthread_once（线程安全）
  • memory barrier 模式（线程安全）
  • Atomic（线程安全）
