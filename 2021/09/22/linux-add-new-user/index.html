Linux 创建新用户

Sep 22, 2021
#Linux

1 常用命令

创建用户：

  • useradd
  • adduser

区别：

  • useradd 需要使用参数选项指定基本设置，如果不使用任何参数，则创建的用户无密码、无主目录、没有指定shell版本。
  • adduser 会自动为创建的用户指定主目录、系统shell版本，会在创建时输入用户密码。
